🎮 Number Guessing Game in C
A simple and fun terminal-based number guessing game written in C.
The program generates a random number between 1 and 100, and the player tries to guess it in the fewest attempts.
🚀 Features
🔢 Random number generation
📉 Hints for each guess (Too high / Too low)
🎯 Tracks number of attempts
🧠 Simple and beginner-friendly logic
💻 Runs in any C-supported environment
🛠️ Technologies Used
C Programming Language
Standard Libraries (stdio.h, stdlib.h, time.h)# Number-Guessing-game-using-c
